LangChain Runnable 실습

@langchain/core/runnables
RunnableSequence
여러 Runnable을 순서대로 연결합니다. pipe()와 동일하지만 배열로 명시적 체인 구성이 가능합니다.
RunnableSequence
const chain = RunnableSequence.from([ prompt, // Step 1: PromptTemplate → PromptValue llm, // Step 2: ChatOpenAI → AIMessage parser, // Step 3: StringParser → string ]); const result = await chain.invoke({ topic });
실행 흐름
최종 출력 (string)
RunnablePassthrough
입력을 그대로 통과시키거나, .assign()으로 기존 객체에 새 키를 추가합니다.
RunnablePassthrough
// .assign()으로 입력 객체에 새 필드 추가 RunnablePassthrough.assign({ question_length: RunnableLambda.from((i) => i.question.length), uppercased: RunnableLambda.from((i) => i.question.toUpperCase()), }) // 입력: { question } → 출력: { question, question_length, uppercased }
assign() 결과 — 입력 객체에 추가된 필드
RunnableLambda
일반 JS 함수를 Runnable로 래핑해 체인에 연결합니다. 동기/비동기 함수 모두 지원합니다.
RunnableLambda
const step1 = RunnableLambda.from((input) => ({ original: input.text, cleaned: input.text.trim().replace(/\s+/g, " "), })); const step2 = RunnableLambda.from((input) => ({ ...input, char_count: input.cleaned.length, word_count: input.cleaned.split(" ").length, })); const chain = RunnableSequence.from([step1, step2, step3]);
Lambda 파이프라인 처리 결과
RunnableParallel
동일한 입력을 여러 체인에 동시에 병렬 실행합니다. 결과는 키-값 객체로 합쳐집니다.
RunnableParallel
const parallelChain = RunnableParallel.from({ definition: makeChain("{topic}을(를) 한 문장으로 정의해주세요."), example: makeChain("{topic}의 실생활 적용 예시를 한 가지만 알려주세요."), future: makeChain("{topic}의 미래 전망을 한 문장으로 말해주세요."), }); // 3개 체인이 동시에 실행 → { definition, example, future } const result = await parallelChain.invoke({ topic });

RunnableBranch
조건 함수에 따라 다른 체인을 실행합니다. if/else 분기 처리에 사용합니다.
RunnableBranch
const branch = RunnableBranch.from([ [(i) => /코딩|코드/.test(i.category), codingChain], [(i) => /수학/.test(i.category), mathChain], [(i) => /과학/.test(i.category), scienceChain], defaultChain, // 기본값 (어떤 조건도 미충족 시) ]); // Step 1: 분류 체인으로 카테고리 판단 // Step 2: 카테고리에 맞는 체인 실행
