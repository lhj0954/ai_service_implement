StringOutputParser
AIMessage 객체에서 .content 문자열만 추출합니다. 가장 기본적인 파서입니다.
StringOutputParser
import { StringOutputParser } from "@langchain/core/output_parsers"; const parser = new StringOutputParser(); const chain = prompt.pipe(llm).pipe(parser); const result = await chain.invoke({ input }); // AIMessage → string

StructuredOutputParser
Zod 스키마로 출력 구조를 정의하고, getFormatInstructions()로 LLM에게 형식을 안내합니다.
StructuredOutputParser
import { StructuredOutputParser } from "@langchain/core/output_parsers"; const parser = StructuredOutputParser.fromZodSchema( z.object({ title: z.string().describe("영화 제목"), director: z.string().describe("감독 이름"), year: z.number().describe("개봉 연도"), genre: z.array(z.string()).describe("장르 목록"), rating: z.number().min(0).max(10).describe("평점"), summary: z.string().describe("줄거리 요약"), }) ); const fmt = parser.getFormatInstructions(); // LLM 지시문 자동 생성 const chain = prompt.pipe(llm).pipe(parser);
getFormatInstructions() — LLM에 전달되는 지시문

CommaSeparatedListOutputParser
콤마로 구분된 LLM 응답 텍스트를 string[] 배열로 파싱합니다.
CommaSeparatedListOutputParser
import { CommaSeparatedListOutputParser } from "@langchain/core/output_parsers"; const parser = new CommaSeparatedListOutputParser(); const fmt = parser.getFormatInstructions(); // → "Your response should be a list of comma separated values..." const chain = prompt.pipe(llm).pipe(parser); const result = await chain.invoke({ category, format_instructions: fmt }); // "김치, 비빔밥, 불고기, ..." → ["김치", "비빔밥", "불고기", ...]
